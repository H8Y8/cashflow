Precision Cashflow | 精準現金流預測模型
這是一個純前端的個人財務現金流預測工具，協助您快速掌握未來的現金水位與風險。
(部署後可替換為您的實際截圖)
✨ 特色
• 隱私安全：所有資料僅儲存於您的瀏覽器 (LocalStorage)，不傳送至任何伺服器。
• 精準計算：支援單次、每月、每季、每年及自訂週期的收支項目。
• 情境模擬：透過開關 (Toggle) 快速啟用或停用項目，即時觀察對未來現金流的影響。
• 視覺化圖表：自動繪製現金水位折線圖，一眼看出資金缺口。
• 資料可攜：支援匯出/匯入 JSON 格式備份。
🚀 如何使用
1. 開啟網頁後，設定目前的「起始現金」與「起始月份」。
2. 點擊「新增」按鈕，加入您的固定收入（如薪水）與支出（如房租、保險）。
3. 右側圖表會即時更新，若線條低於零軸，系統會發出警告。
4. 可以嘗試關閉某些支出項目，模擬省錢計畫對未來的影響。
🛠️ 技術細節
• 架構：純 HTML5 + CSS3 + ES6 JavaScript (Single File Component)
• 樣式：Tailwind CSS (透過 CDN 載入)
• 圖標：Phosphor Icons
• 設計語言：Precision Edge (0px 圓角、銳利邊界、專業風格)
